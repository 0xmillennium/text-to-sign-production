# ADR-0004: English As Project Language

- Status: Accepted
- Date: 2026-04-04

## Context

The thesis repository may be shared with collaborators, examiners, and future open-source
readers. Mixed-language artifacts would reduce consistency and accessibility.

## Decision

All code, comments, documentation, developer-facing text, ADRs, and experiment logs will be
written in English.

## Consequences

- Repository materials remain consistent and easier to review.
- Future publication and collaboration pathways stay simpler.
- Contributors must translate or rewrite non-English draft material before merging it.
