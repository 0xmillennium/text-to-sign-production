# Experiment Log Template

## Objective

State the research question or engineering hypothesis.

## Dataset / Version

Identify the dataset source, subset, and version or revision used.

## Config

List the relevant configuration values, files, and environment assumptions.

## Command

Record the exact command or notebook entry point used to run the experiment.

## Result Summary

Summarize the main outputs, metrics, and interpretation.

## Notes

Capture anomalies, caveats, or operational details.

## Next Step

State the most important follow-up action.
